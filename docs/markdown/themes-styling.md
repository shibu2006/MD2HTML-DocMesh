## Code Syntax Highlighting

When "Highlight Code" is enabled, fenced code blocks receive language-specific coloring.

### Supported Languages
Highlight.js provides syntax highlighting for 190+ languages including:
- JavaScript, TypeScript, Python, Java
- HTML, CSS, JSON, YAML
- Bash, SQL, Markdown
- And many more

### Usage
Specify the language after the opening fence:

~~~markdown
```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
~~~

## Typography

### Heading Styles
- Headings use the theme's accent color
- Bold weight for emphasis
- Proper hierarchy sizing (h1 > h2 > h3...)

### Body Text
- Comfortable line height (1.6)
- Theme-appropriate text color
- Proper paragraph spacing

### Code Blocks
- Monospace font
- Theme-matched background
- Subtle border for definition
- Horizontal scrolling for long lines
