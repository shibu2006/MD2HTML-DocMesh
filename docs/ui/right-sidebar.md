# Right Sidebar

The right sidebar provides settings and controls, adapting to the current mode.

## Markdown Mode

### Export Settings Section

**Output Format**
- HTML5 Complete - Full document
- HTML Fragment - Content only

**Theme**
Dropdown with 7 theme options affecting colors and styling.

**Font Family**
Dropdown with 7 font options from system to monospace.

**Font Size**
Dropdown with 4 size options (14px to 20px).

### Feature Toggles

**Include Table of Contents**
- Toggle switch to enable/disable
- Position buttons appear when enabled:
  - Left Sidebar
  - Top of Page

**Sanitize HTML**
Toggle for DOMPurify security filtering.

**Include CSS**
Toggle to embed styles in export.

**Minify Output**
Toggle to remove whitespace.

**Highlight Code**
Toggle for syntax highlighting.

## DocMesh Mode

### Mesh Controls
See [Mesh Controls](../docmesh/mesh-controls.md) for details.

- New Mesh button
- Save Mesh button
- Export Mesh button
- Load saved mesh dropdown
- Current mesh info card

### Preview Settings

**Preview Mode Toggle**
- Node View - Single document
- Index View - Full navigation

### Node Settings
Appears when a node is selected:

**Display Title**
- Text input for custom title
- Shows default (filename) when empty

**Description**
- Textarea for navigation description
- Optional, shown in index navigation

### Export Settings Note
Reminder that markdown export settings apply to DocMesh exports.

---

## Status Card

Appears in both modes at the bottom:

- **Version** - Current app version (1.1.0)
- **Status** - Ready indicator (green)
