# Quick Start Guide

Get up and running with MD2HTML DOCMesh in minutes.

## First Steps

### 1. Upload Your First Markdown File

- Click the **Upload Files** button in the left sidebar
- Or drag and drop `.md` files directly onto the application
- Supported formats: `.md`, `.markdown`

### 2. Edit Your Content

- Select a file from the file list to make it active
- Click the **Editor** tab in the toolbar to edit markdown
- Your changes are saved automatically in the session

### 3. Preview Your HTML

- Click the **Preview** tab to see rendered HTML
- The preview updates in real-time as you edit
- Styling reflects your current export settings

### 4. Export to HTML

- Click **Download HTML** to export the current file
- Click **Download All** to export all files as a ZIP archive

## Switching Modes

Use the mode toggle in the left sidebar to switch between:

- **Files** - Markdown editing mode
- **DocMesh** - Document organization mode

## Theme Toggle

Click the sun/moon icon in the header to switch between light and dark UI modes.
