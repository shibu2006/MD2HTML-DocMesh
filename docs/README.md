# MD2HTML DOCMesh Documentation

Welcome to the MD2HTML DOCMesh documentation! This guide will help you understand and use all features of the application.

## What is MD2HTML DOCMesh?

MD2HTML DOCMesh is a powerful markdown-to-HTML converter and document organization web application. It allows you to:

- Convert markdown files to beautifully styled HTML
- Organize multiple HTML documents into navigable documentation sites
- Apply custom themes and styling to your exports
- Create hierarchical document structures with drag-and-drop

## Application Modes

The app operates in two main modes:

1. **Markdown Mode** - Edit, preview, and export markdown files to HTML
2. **DocMesh Mode** - Organize HTML files into hierarchical documentation sites
